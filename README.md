# Soul Tech — Site Institucional

Este repositório contém o código-fonte do site institucional da **Soul Tech**, uma startup de tecnologia focada no desenvolvimento de sistemas internos, plataformas digitais e automações sob medida.

## Sobre a Soul Tech

A Soul Tech nasce com o propósito de usar tecnologia para resolver problemas reais de negócio.
Não criamos software apenas por criar — entendemos processos, dores e objetivos antes de escrever qualquer linha de código.

Atuamos com uma mentalidade de produto, aplicando nos projetos dos clientes os mesmos princípios usados na criação de soluções escaláveis próprias.

## O que fazemos

- Sistemas internos sob medida  
- Plataformas digitais e aplicações web  
- Automação de processos e integrações  
- Consultoria técnica e definição de arquitetura  
- Evolução e sustentação de soluções digitais  

## Princípios da Soul Tech

- O problema do cliente vem antes do software  
- Simplicidade é uma escolha estratégica  
- Tecnologia precisa ser fácil de usar  
- Inovação com propósito, não por hype  
- Crescimento sustentável acima de crescimento rápido  
- Cada projeto deve ter alma, ética e responsabilidade  

## Objetivo do site

O site institucional tem como objetivo:
- Apresentar a proposta de valor da Soul Tech  
- Comunicar serviços, cultura e diferenciais  
- Servir como canal de contato com empresas e parceiros  
- Atuar como vitrine da capacidade técnica da empresa
